# Control Center Roadmap

## Objetivo
Consolidar NEXORA Control Center como painel central de governança da plataforma.

## Menu alvo
- Dashboard
- Platform
- Apps
- AI
- Memory
- Skills
- Orchestrator
- Runtime
- Events
- Logs
- Permissions
- Marketplace
- Settings

## Entrega por etapa
- Etapa atual: Dashboard, Platform e Apps funcionais.
- Próxima etapa: Runtime, Events e Logs com dados reais de operação.
- Etapa seguinte: Permissions, Marketplace e Settings com governança avançada.

## Requisitos de produto
- Visão global da saúde da plataforma.
- Navegação por app com diagnóstico operacional.
- Governança de skills e orquestração.
- Rastreabilidade completa por tenant, app e usuário.

## Dependências
- App Registry estável.
- APIs de plataforma versionadas.
- SDK para onboarding de novos apps.
